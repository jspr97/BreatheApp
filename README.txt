1. Copy or clone the link of the project.
2. Open android studio.
3. Select "Check Out project from Version Control" option.
4. Select Git from the dropdown menu
5. Paste the link of the project into the url.
6. Press clone.
7. Load the project.
8. Run the project in emulator or any android phone.
9. Use gmail to sign in into the application.
10. Enjoy!
